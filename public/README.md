# Public Assets Directory

## Logo Image Setup

To add your TruVixo logo to the website:

1. **Save your logo image** in this folder as `logo.png`
   - Recommended size: 48x48 pixels (minimum) or 96x96 pixels (optimal)
   - Supported formats: PNG, JPG, SVG
   - File name must be exactly: `logo.png`

2. **Logo requirements:**
   - Should have a transparent or white background
   - Should be square or close to square aspect ratio
   - Should be high quality and crisp
   - Should work well on both light and dark backgrounds

3. **After adding the logo:**
   - The logo will automatically appear in the navbar and footer
   - The logo will be optimized by Next.js for performance
   - The logo will be responsive and scale appropriately

## Current Logo Usage

- **Navbar**: Logo appears next to "TruVixo" text
- **Footer**: Logo appears in the company info section
- **Size**: 48x48 pixels (w-12 h-12 in Tailwind)

## Alternative Logo Names

If you prefer a different filename, you can also use:
- `logo.jpg`
- `logo.svg`
- `truvixo-logo.png`

Just make sure to update the `src` attribute in the components accordingly.
