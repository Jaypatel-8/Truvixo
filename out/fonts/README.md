# Font Files Setup

To use the "Le Petite Cocho" font, please add the following font files to this directory:

## Required Font Files:
- `LePetiteCocho-Regular.woff2`
- `LePetiteCocho-Regular.woff`
- `LePetiteCocho-Regular.ttf`
- `LePetiteCocho-Bold.woff2`
- `LePetiteCocho-Bold.woff`
- `LePetiteCocho-Bold.ttf`

## Font Formats:
- **WOFF2**: Modern, compressed format (preferred)
- **WOFF**: Fallback for older browsers
- **TTF**: TrueType format for maximum compatibility

## How to Add Fonts:
1. Download or obtain the "Le Petite Cocho" font files
2. Place them in this `/public/fonts/` directory
3. Ensure the filenames match exactly as listed above
4. The website will automatically use these fonts once they're in place

## Fallback Fonts:
If the font files are not available, the website will fallback to:
- Inter (primary)
- Space Grotesk (display)
- System fonts

The font system is already configured and ready to use "Le Petite Cocho" as soon as the font files are added!
